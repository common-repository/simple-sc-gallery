A simple plugin that uses the shortcode [simple-sc-gallery ids="1,2,3"], where ids are the IDs of the images in your WordPress media library.

== Description ==

The Simple SC Gallery simplifies the process of adding image galleries to your WordPress website. Use the shortcode to display comma-separated image IDs from your media library. Customize thumbnail dimensions, effects, and more. Enhance user experiences with responsive galleries that load efficiently.

== Installation ==

1. Log in to your WordPress admin dashboard.
2. Navigate to the "Plugins" section.
3. Click on "Add New."
4. Search for "Simple SC Gallery."
5. Click "Install Now" next to the plugin.
6. Once installed, click "Activate."
7. After activation, youâ€™ll find a new menu item called "Simple SC Gallery" in your WordPress admin menu.
8. Click on it to access the plugin settings and create your image galleries.

== Frequently Asked Questions ==

= How do I add a gallery to my page? =
Use the shortcode [simple-sc-gallery ids="1,2,3"] where ids are the IDs of the images in your WordPress media library.

= How do I customize the appearance of the gallery? =
Navigate to the "Simple SC Gallery" settings page in your WordPress admin dashboard to customize thumbnail dimensions, effects, and other settings.
